## [1.1.0] - 2026-06-29
### 変更
* フッター（`Footer.tsx`）を「H3 Incover Network」の紹介フッターへ全面リニューアル。3カラム化し、公式サイト・INFO・介護制度ポータル・介護ホームページサービスへの回遊カード（ホバーアニメーション付き）を配置。
